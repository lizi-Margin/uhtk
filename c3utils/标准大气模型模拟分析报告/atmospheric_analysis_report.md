## 执行摘要 / Executive Summary

本报告基于国际标准大气模型（ISA），模拟计算了0-50,000英尺高度范围内的大气特性，并绘制了声速与高度的关系图以及等效表速与高度的关系图。分析结果显示，随着高度增加，声速逐渐减小，等效表速显著降低。

This report uses the International Standard Atmosphere (ISA) model to simulate atmospheric characteristics from 0-50,000 ft altitude, plotting speed of sound vs altitude and equivalent airspeed vs altitude. Results show decreasing speed of sound and significantly reduced equivalent airspeed with increasing altitude.

## 物理原理解释 / Physical Principles

### 声速公式
声速在空气中由以下公式给出：

```
c = √(γ × R × T)
```

其中：
- c: 声速 (m/s)
- γ: 比热比 (1.4 for air)
- R: 气体常数 (287.05 J/kg·K for air)
- T: 绝对温度 (K)

**影响声速的因素**:
1. **温度**: 声速与温度的平方根成正比
2. **高度**: 高度增加导致温度降低，声速减小
3. **大气层**: 对流层温度递减，平流层温度恒定

### 等效表速计算
等效表速（Equivalent Airspeed, EAS）与真实空速（True Airspeed, TAS）的关系：

```
EAS = TAS × √(ρ / ρ₀)
```

其中：
- ρ: 当前高度空气密度
- ρ₀: 海平面标准密度 (1.225 kg/m³)

**影响表速的因素**:
1. **密度**: 密度随高度指数衰减
2. **压力**: 压力变化影响密度
3. **温度**: 温度变化影响密度

## 大气特性变化趋势 / Atmospheric Characteristic Trends

### 温度变化
- **对流层**: 温度随高度线性递减，递减率约为 -2°C/1000ft
- **平流层**: 温度保持恒定在 -56.5°C

### 气压变化
- 气压随高度呈指数衰减
- 在36,089英尺处气压约为海平面的22.3%
- 在50,000英尺处气压约为海平面的11.4%

### 密度变化
- 密度变化最显著，随高度指数衰减
- 密度是影响飞行器性能的关键参数
- 直接影响发动机推力和升力产生
